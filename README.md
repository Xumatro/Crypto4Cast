# CryptoPredix
Predict the future trajectory of cryptocurrencies

## How to use
#### Depencencies (install with pip):
* json
* requests
* pandas
* keras
* tensorflow
* silence_tensorflow (optional)

#### Execution:
* ##### Training:
python3 main.py train
* ##### Predicting:
python3 main.py

#### Settings:
##### All setting can be changed in settings.json
